Cara Detail Penggunaan Aplikasi

  1. Isi Informasi Acara: Mulailah dengan mengisi detail acara seperti Nama Acara, Tema, Pembicara, dan lainnya. Informasi ini akan digunakan untuk memberi konteks pada ringkasan AI.
  2. Kumpulkan Catatan: Anda bisa mengumpulkan catatan dari berbagai sumber:
    • Klik "Mulai Rekam Suara" untuk mentranskripsi percakapan secara langsung.
    • Gunakan tombol "Ambil Foto" atau "Ambil Screenshot" untuk menangkap informasi visual.
    • Ketik catatan penting di kolom catatan manual dan klik "Tambah Catatan".
  3. Tinjau Transkrip Gabungan: Semua input Anda (suara, gambar, teks) akan muncul secara real-time di kolom "Gabungan Transkrip".
  4. Buat Ringkasan: Setelah semua informasi terkumpul, klik tombol "Buat Ringkasan". AI akan menganalisis semua data dan membuat notulensi yang terstruktur di bagian "Hasil Ringkasan".
  5. Simpan & Ekspor: Gunakan tombol-tombol di bagian "Simpan Laporan" untuk mengunduh hasil kerja Anda dalam format yang Anda inginkan (PDF, TXT, MD, Audio, atau ZIP berisi foto).

Frequently Asked Questions (F.A.Q)

1. Apakah data saya aman dan disimpan di server?

Privasi Anda adalah prioritas kami. Kami tidak menyimpan data input Anda (audio, gambar) secara permanen. Data hanya diproses sementara untuk menghasilkan output, lalu dihapus. Lihat Kebijakan Privasi kami untuk detailnya.

2. Format audio apa yang didukung?

Aplikasi ini merekam audio dalam format WebM dengan codec Opus, yang efisien dan didukung secara luas oleh browser modern. Saat ini, kami belum mendukung unggah file audio eksternal.

3. Mengapa hasil transkripsi terkadang tidak akurat?

Akurasi transkripsi bergantung pada beberapa faktor, seperti kualitas mikrofon, kebisingan latar belakang, dan kejelasan pembicara. Untuk hasil terbaik, gunakan di lingkungan yang tenang dengan suara yang jelas.

4. Apakah ada batasan durasi rekaman?

Secara teknis tidak ada batasan durasi, namun ada batasan ukuran file sekitar 24 MB untuk menjaga performa. Aplikasi akan memberi peringatan dan berhenti otomatis jika mendekati batas tersebut. Untuk sesi yang sangat panjang, disarankan untuk melakukan beberapa kali rekaman.

5. Bisakah saya mengedit transkrip sebelum diringkas?

Saat ini, fitur untuk mengedit transkrip secara langsung belum tersedia. Namun, Anda dapat menyalin teks dari kotak transkrip, memperbaikinya di editor teks lain, lalu menambahkannya kembali sebagai "Catatan Manual".

Detailed Application Guide

  1. Fill In Event Information: Start by filling in the event details such as Event Name, Theme, Speaker, etc. This information will be used to provide context for the AI summary.
  2. Collect Notes: You can gather notes from various sources:
    • Click "Start Voice Recording" to transcribe conversations live.
    • Use the "Take Photo" or "Take Screenshot" buttons to capture visual information.
    • Type important notes in the manual note field and click "Add Note".
  3. Review the Combined Transcript: All your inputs (voice, image, text) will appear in real-time in the "Combined Transcript" column.
  4. Create Summary: Once all information is collected, click the "Create Summary" button. The AI will analyze all the data and generate structured notes in the "Summary Result" section.
  5. Save & Export: Use the buttons in the "Save Report" section to download your work in your desired format (PDF, TXT, MD, Audio, or a ZIP file of photos).

Frequently Asked Questions (F.A.Q)

1. Is my data secure and stored on the server?

Your privacy is our priority. We do not permanently store your input data (audio, images). Data is only processed temporarily to generate the output and is then deleted. See our Privacy Policy for details.

2. What audio format is supported?

The application records audio in WebM format with the Opus codec, which is efficient and widely supported by modern browsers. Currently, we do not support uploading external audio files.

3. Why is the transcription sometimes inaccurate?

Transcription accuracy depends on several factors, such as microphone quality, background noise, and speaker clarity. for best results, use the app in a quiet environment with clear speech.

4. Is there a limit to the recording duration?

Technically, there is no duration limit, but there is a file size limit of around 24 MB to maintain performance. The app will provide a warning and stop automatically if it approaches this limit. For very long sessions, it is recommended to make multiple recordings.

5. Can I edit the transcript before summarizing?

Currently, a feature to directly edit the transcript is not available. However, you can copy the text from the transcript box, correct it in another text editor, and then add it back as a "Manual Note".
